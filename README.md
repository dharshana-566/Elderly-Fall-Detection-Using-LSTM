# 🧓 Elderly Fall Detection Using LSTM

This project implements an **LSTM-based deep learning system for detecting falls in elderly people** using time-series sensor data.  
It supports **model training**, **command-line inference**, and an **interactive GUI application**.

---

## 📌 Features

- LSTM-based sequence modeling for fall detection  
- Bidirectional LSTM architecture  
- Training, validation, and testing pipeline  
- Tkinter-based GUI for inference on CSV files  
- Supports CPU and GPU (CUDA)  
- Outputs fall probabilities and predictions
